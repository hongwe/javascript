# L5 My Family Tree

#### 正文

    This is my mum.
    This is my dad.
    This is my sister.
    This is my brother.
    This is my grandparents.
    And these are my other grandparents!

#### 单词

    * mum => * dad => * sister <==> * brother
    grandparents => grandfather
    * this <--> * these
    coat -> dress
    dog -> bird
    table -> sofa
    book
    sea - > sand -> lawntree

#### 配音

    Cramp. Cramp.
    Look at that!
