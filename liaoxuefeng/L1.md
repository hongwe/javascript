# 快速入门
JavaScript代码可以直接嵌在网页的`任何地方`,不过通常代码放到`<head>`中

```html
<html>
<head>
  <script>
    alert('Hello, world');
  </script>
</head>
<body>
  ...
</body>
</html>
```

外部引用
```html
<script src="/static/js/abc.js"></script>
```

`type="text/javascript"`不用写
