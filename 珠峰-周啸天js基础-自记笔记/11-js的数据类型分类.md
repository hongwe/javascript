### JS 的数据类型

-   基本数据类型(值类型/原始值)
    -   数字 number
        -   常规数字和 NaN
    -   字符串 string
        -   所有单引号,双引号,反引号(撇)包起来的都是字符串
    -   布尔 boolean
        -   true/false
    -   空对象指针 null
    -   未定义 undefined
    -   ES6新增唯一值 symbol
-   引用数据类型
    -   对象数据类型 object
        -   {} 普通对象
        -   [] 数组对象
        -   /^[+-]?(\d|([1-9]\d+))(\.\d+)?\$/ 正则对象
        -   Math 数学函数对象
        -   日期对象
        -   ...
    -   函数数据类型 function
