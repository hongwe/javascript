  #### JS组成的三部分

  - ECMAScript(ES3/ES6-9) 定义JS的语法规范,定义了语言本身的变量,数据值,操作语句,内存管理...
  - DOM(document object model)
  - BOM(browser object model)

注意当代项目开发,一般VUE/REACT 完成,已经不去操作DOM了,操作数据,框架本身完成DOM操作
